## [0.0.8] - 2026-02-24

- Use absolute image paths

## [0.0.7] - 2026-02-24

- Fixed broken screenshot images (converted from submodule to regular directory)

## [0.0.6] - 2026-02-23

- Fixed broken screenshot images by marking PNGs as binary in .gitattributes

## [0.0.5] - 2026-02-23

- Added README with installation instructions and screenshots
- Added screenshot images for TSX, TypeScript, HTML, CSS, JSON, Python, and C#

## [0.0.4] - 2026-02-23

- Added Fira Code Light font for comments
- Added semantic token colors (semanticHighlighting: true)
- Added editor errors/warnings/info colors
- Added editor gutter colors (git added/modified/deleted)
- Added editor overview ruler colors
- Added minimap colors and gutter indicators
- Added terminal ANSI colors (all 16)
- Added diff editor colors
- Added merge conflict colors
- Added editor extras (fold, sticky scroll, code lens, light bulb, etc.)
- Added debug console and debug icon colors
- Added menu and menubar colors
- Added testing icon colors
- Added keybinding label colors

- Updated built-in/library function colour
